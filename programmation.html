Commentaire sur le language html

Ce language est pratique pour faire des pages web mais, il est difficile de si retrouver car, pratiquement tou ce fait par block et non par ligne de commande. Cependant, son utilité est présente car, ce language se concentre sur le fait qu'on travaille sur un interface graphique. Personnellement, je n'aime pas le fait qu'on prend le double d'espace afin de faire une action la plupars du temp. Donc, je crois qu'il y a moyen de raffiné se language encore plus dans les années a venir.

Commentaire sur le language C#

Personnellement, Je suis en amour avec ce language car, il y moyen de raccourcir le code en passant tout dans des fonctions qui retourne un résultat. Donc, dans la parti principale du programme, on se retrouve avec seulement les actions voulus et on peut facilement comprendre ce qui ce passe a l'écran.
